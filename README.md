# tutorial-git

1. 下載專案下來
    ```bash
    git clone <url>
    ```
2. 查看git狀態
    ```bash
    git status
    ```
3. 放到暫存區
    ```bash
    git add .
    # 或者
    git add <檔案名>
    ```
4. 確認修改
    ```bash
    git commit -m "修改的訊息"
    ```
5. 查看紀錄
    ```bash
    git log
    ```
5. 上傳到github
    ```bash
    git push origin master
    ```
6. 取得最新版本
    ```bash
    git fetch
    ```
7. 取得最新版本並且合併
    ```bash
    git pull origin master
    ```

程式碼 -> 暫存區 -> 確認修改
